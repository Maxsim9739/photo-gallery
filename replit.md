# Фотогалерея

Веб-приложение для хранения и просмотра фотографий с ролевой системой (редактор / гость).

## Стек

- **Node.js** + **Express** — сервер
- **EJS** — шаблонизатор
- **multer** — загрузка файлов
- **express-session** — сессии
- **bcryptjs** — хэширование паролей

## Запуск

Workflow **Start application** запускает `node server.js` на порту 5000.

## Переменные окружения

| Переменная      | Описание                          | По умолчанию |
|-----------------|-----------------------------------|--------------|
| `PORT`          | Порт сервера                      | `5000`       |
| `ADMIN_USER`    | Логин редактора                   | `admin`      |
| `ADMIN_PASS`    | Пароль редактора                  | `admin123`   |
| `SESSION_SECRET`| Секрет сессии (Replit Secret)     | —            |
| `NODE_ENV`      | Окружение (`production` и т.д.)   | —            |

## Структура

```
server.js        — основной сервер
views/
  gallery.ejs    — главная страница галереи
  login.ejs      — страница входа редактора
uploads/         — загружённые фотографии (сбрасываются при перезапуске)
Package.json     — оригинальный манифест из репозитория
package.json     — манифест, используемый npm
```

## Заметки

- Фотографии хранятся **в памяти** — при перезапуске сервера сбрасываются.
- В `Package.json` (оригинал) и `package.json` (Replit) разные наборы зависимостей. npm использует строчный `package.json`.
- `ejs` был добавлен в `package.json` вручную, так как в оригинале он отсутствовал.
